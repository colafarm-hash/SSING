# 3. 서비스 전략

## 3.1 서비스 컨셉 및 방향성

> "17만원에서 5만원으로, 15스텝에서 원클릭으로."

SSING은 스키 강습 시장의 다단계 중개 거품을 제거하고, 요청 즉시 동시간대 가용 강사 풀을 제시하여 소비자가 직접 비교·선택하도록 하는 양면 매칭 플랫폼이다.

| 축 | 방향성 |
|---|---|
| 소비자 측 | 15 step의 탐색·비교를 요청 한 번으로 압축, 가용 강사 풀에서 직접 비교·선택 |
| 강사 측 | "가용" 상태 ON만으로 자동 노출, 행정·스케줄 부담 제거, 등급제 기반 개인 브랜드화 |
| 시장 측 | 70.3% 중개 마진 → 무료화로 시장 구조 재편 |

## 3.2 핵심 목표 (KPI)

> ⚠️ 구체 수치는 추후 확정 필요

### 북극성 지표 (North Star Metric)

시즌 내 매칭 완료 건수 — 소비자·강사 양면의 가치 교환이 실제로 일어났는지를 가장 직접적으로 보여주는 지표.

### KPI 후보군

| 영역 | 지표 | 비고 |
|---|---|---|
| 매칭 효율 | 평균 매칭 소요 시간, 풀 노출 → 선택 전환율 | 목표 매칭 시간: 평균 32초 이내 |
| 소비자 | MAU, 매칭 완료율, 재이용률 | |
| 강사 | 활성 강사 수, 강사 잔존율, 평균 강사 수익, 가용 시간당 매칭률 | |
| 품질 | 강습 만족도(평점), 재예약률 | 등급제 산정의 핵심 변수 |
| 비즈니스 | GMV, 대체 BM 수익 | 무료화로 인해 재정의 필요 |

## 3.3 비즈니스 모델 ⚠️ 재작성 필요

### 전면 무료화 결정

기존의 등급별 차등 Take Rate (5%~20%, 가중평균 13.5%) 모델은 **전면 무료화 (수수료 0%)** 로 전환되었다.

### 대체 수익 모델 후보 (🟡 검증 필요)

- [ ] 광고 매출 잠재력
- [ ] 결제 수수료 마진
- [ ] 장비 중고거래 수수료
- [ ] 프리미엄 강사 멤버십
- [ ] B2B 솔루션 (사설팀 운영 SaaS)

> 무료화 결정 이후의 BM 시뮬레이션은 별도 작업으로 진행 필요. 기존의 "건당 평균 24,000원 / 3년차 9.3억" 수치는 무효.

## 3.4 출시 전략 및 시즌 사이클

### 출시 전략: MVP → 확장 로드맵

| Phase | 핵심 목표 | 주요 기능 |
|---|---|---|
| Phase 1: MVP | 단일 스키장 검증, 콜드 스타트 해결 | 가용 풀 매칭, 강사 프로필, 결제 |
| Phase 2: 확장 | 6대 스키장 커버, 등급제 가동 | 강사 스케줄 관리, 등급제·가격 슬라이더, 누적 평가 |
| Phase 3: 인접 시장 | 비수기 매출 확보, 수직 확장 | 시즌권 사전 예약, 장비 중고장터 |

### MVP 진입 스키장 우선순위 (확정)

**1순위**: 비발디 · 곤지암 · 하이원
**2순위**: 휘닉스 · 엘리시안
**제외**: 용평 (사설강습 공식 불허)

> 콜드 스타트 전략을 위해 1순위 3곳 중 **1곳 집중 진입** 검토 중. ← 🔴 다음 결정 사항

---

### 연간 시즌 사이클

스키 강습은 시즌 비즈니스(12~3월 집중)지만, 비수기를 운영-성장 분기로 명확히 분할하여 연중 사이클을 설계한다.

| 구간 | 기간 | 활동 |
|---|---|---|
| 시즌 운영기 | 12~3월 (3개월) | 매칭 운영, 매출 집중 발생 |
| 시즌 직후 | 4월 | 중고장터 활성화 (장비 처분 수요) |
| 비수기 성장기 | 4~7월 (4개월) | 제품 개선, 강사 풀 확장, 다음 시즌 준비 |
| 시즌 사전 예약 | 8~11월 (4개월) | 시즌권·강습 패키지 사전 예약 오픈 |

> **핵심 명제**: 시즌 매출이 비수기 burn rate를 충분히 커버할 수 있는 구조로 설계한다.
> **선례**: 어프로치(골프)·스매시(테니스)·웨딩북(웨딩) 모두 시즌·이벤트성 시장에서 검증된 모델.

### 장기 확장 가능성

숨고가 단일 레슨 매칭에서 1,000개 카테고리로 확장한 것처럼, SSING도 스키 강습 → 인접 윈터 스포츠 → 시즌형 레슨 시장 전반으로 확장 가능하다.
